Site building basics

Start from scratch with HTML, CSS & JavaScript

About

We had a (hopefully : ) awesome meetup to go through standing up a basic webpage. We built a html page from scratch. We tied in a css page and built out some styles. We added in a javascript page and got everything connected and up and running.

This page is a resource for you to go back and grab any parts you missed. You can also check out some variations in the style sheets and add to your own project. In the future, we might consider using this as a jumping off point for future topics

You can find the source code from this meetup on the Github repo: https://github.com/susanwands/Site-Building-Basics

Themes

Each link here is using the same html. The changes are through the CSS. If you would like to add a theme that you created using the same index page feel free to add to this project. You can pull the file "base.html" to make your theme, just don't make any changes to it except linking your own css and js file to it (no cheating : ) .

  • https://youtu.be/PVzxgYT332I - Session Video
  • http://www.layoutit.com - a tool for playing around with Bootstrap layouts
  • http://getbootstrap.com - awesome documentation and examples for Bootstrap.
  • https://bootswatch.com/ - downloadable Bootstrap templates for quick projects
  • https://cdnjs.com/libraries/modernizr - Modernizr is one of many libraries to help with outdated browser support
  • https://www.w3schools.com/html/default.asp - Quick HTML5 reference
  • https://css-tricks.com/semantic-class-names/ - Great site overall, but this article is on semantic tags and why
  • https://developer.mozilla.org/en-US/docs/Learn/Accessibility/HTML - Mozilla (makers of Firefox browser) have awesome learning resources. This lets you use a screen reader to experience the difference between good, accessible HTML5 and bad code.
